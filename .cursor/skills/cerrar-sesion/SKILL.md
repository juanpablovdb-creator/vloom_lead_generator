# Cerrar sesión

## Cuándo aplicar

Cuando el usuario se despida, indique que va a cerrar, o diga que se va (adiós, hasta luego, cierro, bye, me voy, closing, leaving, etc.).

## Acción

Recordar al usuario el checklist de cierre antes de irse:

```
Antes de cerrar, recuerda:

1. **Probar:** `npm run dev` — verifica que todo funcione
2. **Limpiar:** `npm run lint:fix` — corrige warnings
3. **Guardar:** `git add .` + `git commit -m "descripción"` — guarda los cambios

Ver docs/PASO_A_PASO.md para más detalle.
```
